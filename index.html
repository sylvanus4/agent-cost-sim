파이프라인 고르면, 라우팅으로 얼마 아끼는지 바로 보여줘요

여러 에이전트를 돌리는 파이프라인에서 모든 단계를 최고급 모델로 쓰는 대신 역할별로 모델을 나누면(탐색은 저비용, 검증만 프론티어) 비용이 얼마나 줄까요? 아래 유형을 눌러보세요.

라우팅이 뭐고, 왜 아껴지나요?

1

에이전트 파이프라인은 모델을 여러 번 호출해요. 자료 검색, 초안 작성, 결과 검증처럼 난이도가 다른 일에요.

2

모든 호출에 최고급 모델을 쓰면, 단순 검색에도 비싼 값을 내죠. 검색·읽기는 최고 지능이 필요 없어요.

3

라우팅 = 일의 난이도에 모델 등급을 맞추기. 탐색은 저비용, 판단·검증만 프론티어.
